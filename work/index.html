Work Related Software

Calcium Signal Extract

Created a Matlab Software package that automatically segments fluorescence image stacks into ROIs based on correlated temporal and spatial dynamics of pixel values. Differential morphology and fluorescence activity of the individual cells is automatically extracted and plotted.

Behavioral Training

Developed a Matlab platform for controlling multiple different behavioral paradigms in terms of stimuli and different modules. Hardware interfacing with RFID sensors, microcontrollers, valves and LEDs. Visual stimuli are generated in multiples ways. Data is analysed and plot in real time.

Object Tracker

Build a Python package that can track multiple (10s - 100s) of objects in a video or a camera stream and define areas where entry of an object is to be monitored using computer Vision libraries.

ScanImage Navigator

Developed a Matlab GUI plugin to read the stepper motors of the microscope stage/objective in a coordinate system. Movement paths can be plotted, recorded and commands be sent to the microscope motors to program the stage movement.
